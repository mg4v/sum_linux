uptime    сколько времени включен компьютер, сколько активных пользователей, load average.
uname     Тип ОС
uname -a  Более поробная информация об ОС и версии ядра.
lspcu     Информация о CPU
ls        вывести содержимое дериктории (по умолчанию - текущей)
echo $PATH    перечень директорий в которых осуществляется поиск исполняемых файлов (программ) при вводе их названий в терминале.
echo "<текст>"    вывести <текст> на экран
CTRL+ALT+F3    открыть виртуальный терминал № 3 (1-6 терминалы, 7 - GUI)
man -k <ключевое_слово>    выводит список команд (имен програм) по клбчевому слову с кратким описанием
man <имя_программы>    открывается полное руководство по команде:
  - PgUp/PgDown    вверх/вниз
  - /<искомый_текст>   искать текст в справке
      n    перейти на следующее вхождение искомого слова
  - Q    выход
CTRL+C    прервать исполнение команды (программы) и выйти из нее.
info <имя_команды>   интерактивная справка в формате html
whatis <имя_команды>    краткое описание команды (ее значение)
whereis <имя_команды>     выводит все каталоги введенной команды (исполняемые файлы, справка, настройки)
locate <файл/команда>    ищет указанный файл/команду
ls -l    подробный вывод содержимого дериктории
CTRL+Z    выйти из программы и оставить в спящем режиме (в фоне)
          fg    (fore ground) - вернуть спящий процесс.
ping <IP/FQDN>    пинг указанного хоста
ps    показать процессы текущего пользователя (в текущем сеансе консоли)
ls -laR <имя_дериктории> - рекрсивно вывести подробный сисок содержимого указанной дерриктории
    включая скрытые файлы и каталоги
sudo  <команда>  (super user do) - выполнить команду с повышенными до супер пользователя правами.
